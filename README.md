## Commit Convention
- [AngularJS Git Commit Message Conventions](https://gist.github.com/stephenparish/9941e89d80e2bc58a153) 을 따른다.
- `feat`: 새로운 기능 개발
- `fix`: 버그 수정
- `refactor`: 기능 변경이 없는 코드 수정
- `test`: 테스트 코드 작성 및 수정
- `docs`: 문서 작성 및 수정
- `build`: 빌드 파일 작성 및 수정
- `style`: 스타일 변경(reformat, indent)
- `chore`: 단순한 작업
